Different Status Examples:

← After application submitted
← After approval by authorities
← Final enrollment status

🔄 How It Works:

  1. Initial State: Button shows "View Details" - Parent can view scheme information and apply
  2. After Application: Button changes to "Approval Pending" with animated pulse dot
  3. After Approval: Button shows "Approved" - Parent knows application was accepted
  4. Final State: Button shows "Enrolled" - Child is officially enrolled in the scheme

Benefits: Clear visual feedback, prevents duplicate applications, shows progress status
